# Shop-Stock
Web application(FE + BE) for placing orders and viewing available products.

Backend build with TypeScript:
-TypeORM
-JWT Authentication
-Express
-class-validator
-cors
-bcrypt

FrontEnd build with Vue.js(vite)
-axios
-bootstrap
-lodash
-vue-router
-vue-sweetalert2
